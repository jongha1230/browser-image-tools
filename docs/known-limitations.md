# Known Limitations And Future Work

## 현재 제한 사항

- 입력과 출력은 JPEG, PNG, WebP만 지원합니다.
- 모든 처리는 브라우저 메모리 안에서 이뤄지므로 매우 큰 파일이나 대량 배치에서는 느려질 수 있습니다.
- 브라우저가 특정 형식 재인코딩을 지원하지 않으면 일부 출력 형식은 실패할 수 있습니다.
- EXIF 제거는 재저장 방식이라 메타데이터 정리에 도움이 되지만, 이미지 안에 보이는 민감 정보까지 숨기지 않습니다.
- 분석 스크립트와 광고 스크립트는 아직 연결하지 않았습니다.

## 출시 후 우선순위가 높은 작업

- 대용량 배치 처리 시 메모리 사용량과 진행률 체감 개선
- 도구별 프리셋과 입력 검증 문구 세분화
- 실제 광고 운영 정보가 준비된 뒤 정책 고지와 관련 기술 파일 보강
- 검색 유입 기준으로 가이드 내부 링크와 FAQ 확장

## 범위 밖 항목

- 백엔드 업로드
- 계정, 인증, 데이터베이스
- PDF, HEIC, RAW, 비디오 처리
- 엔드유저 대상 AI 기능
